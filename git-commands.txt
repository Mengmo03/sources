# Git 命令 - 推送项目到 GitHub
# 仓库地址: https://github.com/Mengmo03/sources.git

# 1. 初始化 git 仓库
git init

# 2. 添加远程仓库
git remote add origin https://github.com/Mengmo03/sources.git

# 3. 添加所有文件到暂存区
git add .

# 4. 提交更改（请根据实际情况修改提交信息）
git commit -m "Initial commit"

# 5. 推送到远程仓库（首次推送）
git push -u origin main

# 如果默认分支是 master 而不是 main，使用：
# git push -u origin master

# 如果遇到推送被拒绝（因为远程仓库已有内容），可以使用：
# git push -u origin main --force
# （注意：--force 会覆盖远程仓库的内容，请谨慎使用）
