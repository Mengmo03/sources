## 使用教程

- 点击**页面左下角**的“![](./requirements/1.png)”按钮即可展开/收起侧边导航栏

- 展开导航栏后，可直接进行全站搜索或点击相关链接跳转到相应页面

!> **注意：**本站所用的搜索技术是基于本地缓存的，其默认刷新时间为一天，因此您搜索到的内容与最新内容可能存在最多一天的误差。<br/>
要解决此问题，您可以使用浏览器的“访客模式”访问本站，以获取最新内容。
